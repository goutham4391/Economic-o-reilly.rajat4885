Integrations

Easier to find. Faster to use. Simpler reporting.

Create the best learning experience for your teams by integrating the O’Reilly learning platform with the tools they already use.

O’Reilly Answers for Slack

View documentation

O’Reilly Answers will respond to your question with highly relevant snippets of content right within Slack.

  • Use the “More Answers” button to flip through the suggested answers
  • Read the full text of an answer in context within the O’Reilly learning platform
  • Share a helpful answer with others in Slack

Single sign-on (SSO)

View documentation

Create a seamless O’Reilly experience for your teams with common identity technologies.

  • Simplifies the login experience and increases productivity
  • Minimizes bad password habits—reducing security risks and help desk costs
  • Accelerates adoption of company-promoted apps

For individually formatted links, see the SSO deep link guide.

Search API

View documentation

Give your teams easy access to all of the O’Reilly learning platform right from your learning experience platform.

  • Updates your LXP with the freshest, most current content
  • Includes descriptions, cover images (metadata), and links to each piece of content for more accurate search results
  • Contains books, audiobooks, videos, live training courses, and more

Insights API

View documentation

Automate a data feed of your teams’ activity on O’Reilly to sync with your own reporting system.

  • Keeps all your reporting together in a single system you’re more familiar with
  • Includes a flexible RESTful API that’s easy to set up and use
  • Pulls in the raw, fine-grain usage data you need to create any custom report
